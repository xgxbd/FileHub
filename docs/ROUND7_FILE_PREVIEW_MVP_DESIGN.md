# Round 7 文件预览 MVP 设计

## 1. 目标

- 在文件中心支持图片、PDF、TXT 的在线预览。
- 不支持类型保持下载路径（不做在线渲染）。

## 2. 范围

- 后端：
  - `GET /files/{file_id}/preview`
  - 权限与下载一致（用户仅预览自己的文件，管理员可预览全部）
  - 支持类型：
    - `image/*`
    - `application/pdf`
    - `text/*`
- 前端：
  - 文件中心新增“预览”按钮
  - 弹层预览：图片、PDF iframe、TXT 文本框

## 3. 响应策略

- 可预览类型：`200`
- 不支持类型：`400`
- 不存在或无权限：`404/403`
- 响应头：`Content-Disposition: inline`

## 4. 最小提交拆分

### 提交 D（前端预览接入）
