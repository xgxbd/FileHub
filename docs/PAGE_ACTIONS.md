# 页面交互说明（方案 D 当前实现）

## 1. 登录页 `/login`

| 控件 | 可点击 | 动作 | 成功反馈 | 失败反馈 |
|---|---|---|---|---|
| 登录按钮 | 是 | 调用 `POST /auth/login` | 跳转到 `/files` 或原重定向地址 | 错误消息（账号/密码错误） |
| 没有账号？去注册 | 是 | 跳转 `/register` | 显示注册页 | 无 |

## 2. 注册页 `/register`

| 控件 | 可点击 | 动作 | 成功反馈 | 失败反馈 |
|---|---|---|---|---|
| 注册并登录 | 是 | 调用 `POST /auth/register`，随后自动登录 | 跳转 `/files` | 显示注册失败原因 |
| 已有账号？去登录 | 是 | 跳转 `/login` | 显示登录页 | 无 |

## 3. 文件列表页 `/files`

### 顶部动作

| 控件 | 可点击 | 动作 | 反馈 |
|---|---|---|---|
| 上传到当前目录 | 是 | 跳转 `/upload?folder=<当前目录>` | 上传页“目标文件夹”自动带入 |
| 回收站 | 是 | 跳转 `/recycle` | 显示回收站 |
| 当前目录标签 | 否 | 仅展示状态 | 无 |

### 文件树区域

| 控件 | 可点击 | 动作 | 反馈 |
|---|---|---|---|
| 文件树节点 | 是 | 切换 `directory` 过滤条件并刷新列表 | 表格仅显示该目录下文件 |
| 刷新 | 是 | 重新加载文件树 | 树节点更新 |
| 手动输入目录 + 切换目录 | 是 | 设置目录过滤并刷新列表 | 当前目录变化 |
| 回到根目录 | 是 | 清空目录过滤 | 显示全部文件 |

### 列表区域

| 控件 | 可点击 | 动作 | 反馈 |
|---|---|---|---|
| 查询 | 是 | 按关键字/大小/目录过滤请求 `GET /files` | 表格刷新 |
| 重置条件 | 是 | 清空关键字/大小条件 | 表格刷新 |
| 预览 | 是 | 跳转 `/preview/{fileId}?name=<file_path>` | 进入预览页 |
| 下载 | 是 | 请求 `GET /files/{id}/download` | 浏览器下载文件 |
| 删除 | 是 | 请求 `DELETE /files/{id}` | 文件进入回收站 |

## 4. 上传页 `/upload`

| 控件 | 可点击 | 动作 | 反馈 |
|---|---|---|---|
| 目标文件夹输入框 | 是 | 设置上传路径前缀 | 最终文件名变为 `folder/filename` |
| 文件选择 | 是 | 选择本地文件 | 显示已选择路径 |
| 开始上传 | 是 | 分片上传流程：创建会话 -> 上传分片 -> 完成合并 | 进度条推进，成功提示“上传完成” |
| 返回文件列表 | 是 | 跳转 `/files` | 返回文件中心 |

## 5. 预览页 `/preview` 与 `/preview/:fileId`

| 控件 | 可点击 | 动作 | 反馈 |
|---|---|---|---|
| 最近文件-预览按钮 | 是 | 跳转 `/preview/{id}?name=<file_path>` 并加载预览 | 主预览区显示图片/PDF/TXT |
| 下载当前文件 | 是 | 请求 `GET /files/{id}/download` | 浏览器下载 |
| 返回文件列表 | 是 | 跳转 `/files` | 返回文件中心 |
| 刷新列表 | 是 | 重新拉取最近文件 | 底部列表更新 |

说明：当前不需要手工输入 fileId，优先从“最近文件（按路径）”选择。

## 6. 回收站页 `/recycle`

| 控件 | 可点击 | 动作 | 反馈 |
|---|---|---|---|
| 查询/重置 | 是 | 按条件过滤回收站 | 列表刷新 |
| 恢复 | 是 | 请求 `POST /recycle/files/{id}/restore` | 文件恢复到正常列表 |
| 彻底删除 | 是 | 请求 `DELETE /recycle/files/{id}/purge` | 文件与对象内容彻底移除 |

## 7. 后台管理页 `/admin/files`（管理员）

| 控件 | 可点击 | 动作 | 反馈 |
|---|---|---|---|
| 查询/重置 | 是 | 管理员条件筛选文件 | 列表刷新 |
| 删除 | 是 | 管理员删除文件到回收站 | 状态变更 |
| 恢复 | 是 | 管理员恢复文件 | 状态变更 |
| 彻底删除 | 是 | 管理员永久删除文件 | 记录消失 |

## 8. 操作日志页 `/admin/logs`（管理员）

| 控件 | 可点击 | 动作 | 反馈 |
|---|---|---|---|
| 查询/重置 | 是 | 请求 `GET /admin/logs` 按条件筛选 | 日志列表刷新 |

## 9. 顶栏全局元素

| 控件 | 可点击 | 动作 | 说明 |
|---|---|---|---|
| 用户名标签 | 否 | 状态显示 | 当前登录用户 |
| 全局搜索规划中 | 否 | 状态显示 | 明确暂未实现，避免误点 |
| 管理员账号 | 否 | 状态显示 | 仅管理员可见 |
| 退出 | 是 | 清理本地会话并跳转登录页 | 安全退出 |
